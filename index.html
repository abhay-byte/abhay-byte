01 // ABOUT

Who I Am

An open source enthusiast and FOSS developer, passionate about community-driven development and making software accessible for everyone.

I'm an Agentic Wizard — I don't just write code, I build the systems that let AI agents write it with me.

My daily workflow is fully agentic. Claude Code, Codex CLI, Gemini CLI, Qwen, Aider, and a rotating cast of AI tools run side by side, each handling what it does best. To orchestrate all of this, I built Nexus Terminal — a multi-agent terminal workspace where multiple AI agents collaborate in parallel.

Beyond the terminal, I work extensively with MCP servers to extend what agents can actually do — giving them real tools, real context, and real capabilities beyond just generating text.

I also run a fleet of specialized bots that keep everything current automatically. The whole system runs on Android through Flux Linux — a full Linux environment, multiple AI agents, continuous background updates — all from a phone.

Every project here is open source. All code, all assets, no paywalls — just FOSS, built in the open.

FEATURED // NEXUS

The Nexus Terminal

A multi-agent AI terminal workspace built with Tauri + React. Run 13+ AI coding agents side-by-side — Claude Code, Codex CLI, Gemini CLI, Qwen, Aider, and more. Available as both a desktop app and a standalone web server you can access from any browser on your network.

  • 13+ AI agents — auto-detected on PATH
  • Split panes — horizontal/vertical 2×2 layouts
  • Session persistence — survives app restarts
  • Kanban board — Todo / In Progress / Done / Blocked
  • True-color PTY — xterm-256color rendering
  • Shared MCP registry — configure once, sync everywhere
  • Brutalist UI — high-contrast dark mode, #ffcc00 accent
